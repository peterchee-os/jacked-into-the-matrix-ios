## Product One-Liner

Jacked into the Matrix is a wearable instruction engine that turns how-to knowledge into glanceable, step-by-step guidance on Even Realities G2 glasses.

## Core Product Thesis

The iPhone does the heavy lifting.
The glasses do the execution.
AI converts messy source content into compact, wearable guidance.

## Starter Use Cases

1. Electrical → Wire a three-way light switch
2. Software / CLI → Update OpenClaw via CLI
3. Climbing / Safety → Pass the lead belay gym test
4. Cooking → Garden fresh pesto pasta
