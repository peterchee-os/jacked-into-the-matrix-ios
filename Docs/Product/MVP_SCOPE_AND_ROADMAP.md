# MVP Scope and Roadmap
## Jacked into the Matrix

## V1 Objective

Ship a curated script player that proves the value of wearable step-by-step guidance on Even G2 glasses.

## In Scope

- Native iPhone app
- Categories
- Favorites
- Recents
- Four curated starter scripts
- Script detail screen
- Step-by-step mode
- Continuous mode
- Drill mode
- Local persistence
- Glasses send / resend / clear flow
- Resume playback state
- Risk labels and warnings

## Out of Scope

- automatic YouTube search and transcript extraction
- local or cloud AI generation as a required dependency
- camera understanding
- voice UI
- script sharing marketplace
- analytics beyond basic product metrics

## Starter Scripts

1. Wire a three-way light switch
2. Update OpenClaw via CLI
3. Pass the lead belay gym test
4. Garden fresh pesto pasta

## Release Phases

### Phase 1: Foundation
- App shell
- Data model
- Local storage
- Home, Categories, Favorites, Recents
- Seed starter scripts

### Phase 2: Script Detail + Playback
- Script detail screen
- Warnings and prerequisites
- Step-by-step mode
- Continuous mode
- Drill mode
- Resume state

### Phase 3: G2 Integration
- Connect to G2 flow
- Send active script
- Next/back synchronization
- Glasses status screen

### Phase 4: Polish
- Editing improvements
- Better error handling
- Risk gating for high-risk scripts
- Performance tuning

## V1 Metrics

- scripts opened
- scripts sent to glasses
- completion rate per script
- favorites added
- repeat usage rate
- most used category
- session length

## Exit Criteria for V1

Ready to move to V2 when:
- users consistently complete tasks with the script player
- playback on G2 feels reliable
- step sizing feels correct for the display
- saved script behavior feels sticky and useful
